# ItemScanner

ItemScanner is a quality-of-life mod for *The Long Dark* that scans nearby items, containers, and harvestable plants and displays markers on screen to assist exploration.

This mod is designed to reduce unnecessary searching while preserving the overall survival experience.

---

## Features

- Scans nearby loose items
- Scans containers such as drawers, cabinets, and lockers
- Scans harvestable plants
- Displays markers for detected objects
- Configurable scan radius and scan interval
- Lightweight and performance-friendly

---

## Requirements

- The Long Dark
- MelonLoader
- ModSettings (**required dependency**)

---

## Installation

1. Install **MelonLoader**
2. Install **ModSettings**
3. Download `ItemScanner.dll` from the Releases page
4. Place `ItemScanner.dll` into the `Mods` folder

---

## Configuration

All settings can be adjusted via **ModSettings** in the in-game Mod Settings menu.

Available options include:
- Scan radius
- Scan interval
- Toggle scanning of items, containers, and plants

---

## Compatibility

- Tested on recent versions of *The Long Dark*
- May require updates after major game patches
