# ItemScanner  
ItemScanner 是一个用于《漫漫长夜（The Long Dark）》的体验优化模组。

ItemScanner is a quality-of-life mod for *The Long Dark*.

---

## Description  
模组会扫描玩家附近的物品、容器以及可采集植物，并在屏幕上显示标记。

The mod scans nearby items, containers, and harvestable plants and displays markers on screen.

该模组旨在减少重复搜索的疲劳感，同时尽量保留原有的生存体验。

This mod is designed to reduce unnecessary searching while preserving the core survival experience.

---

## Features  
- 扫描附近的可拾取物品  
- 扫描抽屉、柜子等容器  
- 扫描可采集植物  
- 显示对应的屏幕标记  
- 可通过设置调整扫描范围与频率  

- Scans nearby loose items  
- Scans containers such as drawers and cabinets  
- Scans harvestable plants  
- Displays on-screen markers  
- Configurable scan radius and scan interval

## Installation  
1. 安装 MelonLoader  
2. 安装 ModSettings  
3. 从 Releases 页面下载 ItemScanner.dll  
4. 将 DLL 放入 Mods 文件夹  

1. Install MelonLoader  
2. Install ModSettings  
3. Download ItemScanner.dll from the Releases page  
4. Place the DLL into the Mods folder  

---

## Configuration  
所有设置均可通过游戏内的 ModSettings 菜单进行调整。

All settings can be adjusted via the in-game ModSettings menu.

可配置内容包括扫描范围、扫描频率以及不同目标的显示开关。

Available options include scan radius, scan interval, and visibility toggles.
